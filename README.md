# Career Pairwise Lab（キャリア比較ラボ）

2択比較を通じて、キャリアの関心・判断軸・避けたい要素を整理するスマホ向けWebアプリです。適職を断定する診断ではなく、次に調べることや相談時の論点を作るためのツールです。

## 使い方

`index.html` をブラウザで開くだけで利用できます。外部API・外部ライブラリ・ビルド工程は不要で、GitHub Pagesにもそのまま配置できます。

## 主な機能

- 現在地とキャリア形態を選び、会社員・組織所属／個人活動のフローへ分岐
- 興味軸、業界・職種または活動ジャンル・収益モデル、仕事スタイルを2択比較
- 任意の比較理由と、避けたい要素・リスクを記録
- 結果を仮説としてランキング表示し、AI分析用プロンプトを生成・コピー
- 結果をブラウザの `localStorage` に最大30件保存し、履歴から再表示

入力内容は外部へ送信されません。保存データは利用中のブラウザ内にのみ保持されます。

## v0.4.1 の改善点

- 質問種別が変わる前に、目的・回答のコツ・進捗を示すセクション遷移画面を追加
- 比較対象の全組み合わせを出題し、ペアワイズランキングの精度を改善
- ランキングにスコア、勝敗数、比較数、勝率、選択理由を表示
- 上位候補が同点または僅差の場合、最大3問のタイブレーク質問を追加
- AI分析用プロンプトと `localStorage` の履歴に、比較統計とタイブレーク結果を保存

## v0.5 の改善点

- 候補を早期除外せず、代表比較・探索比較・接戦比較・反証比較を組み合わせる探索型比較エンジンへ変更
- 各レイヤーを最大8候補・12問に抑え、傾向適合60%・カテゴリ分散25%・探索15%を目安に出題
- 候補ランキングとは別に16種類のtraitスコアを保持し、内部傾向を推定
- 「当たり前品質」と「魅力的品質」のチェック項目をコードと保存履歴に追加
- 会社員・個人活動・創作・副業を横断する可能性マップを結果画面に追加
- trait相性は高いが比較が少ない「未探索の可能性」を表示
- 結果を鵜呑みにしないための反証質問と、次に試す小さな行動を追加

選択したキャリア形態は比較フローの分類に使いますが、候補自体の存在を消す条件には使いません。結果画面では選択モード外の可能性も参考候補として確認できます。

## v0.6 の改善点

- 体験時間を約5分、比較質問を最大20問に制御
- ウォームアップ3問、探索8問、仕事スタイル5問、接戦・反証4問の時間予算型フローへ変更
- 比較途中から現在までの回答で結果を生成できる「ここまでで結果を見る」を追加
- 上位8 traitを表示するSVGレーダーチャートを追加
- 「組織所属↔個人自律」「安定運用↔創作・挑戦」のSVGキャリア可能性マップを追加
- 総合、会社員、個人活動、創作・副業の上位候補を横棒で視覚化
- 「どちらもあり」「わからない」や僅差をまとめる迷い・接戦カードを追加
- 比較数が少なくtrait相性が高い候補を未探索候補カードとして表示

候補マスタの広さは維持し、実際の質問だけを情報量と時間予算に基づいて選びます。途中結果には、回答数が少ない仮説である旨を明示します。

## v0.7 の改善点

- 約10分でより深く探索する体験へ調整し、比較質問を最大36問に拡張
- ウォームアップ5問、探索14問、仕事スタイル8問、接戦・反証6問、タイブレーク最大3問へ変更
- 探索比較では、未知の探索候補を含む質問を最低3問確保
- 会社員・個人活動・創作・副業を横断するウォームアップと反証比較を追加
- v0.6のtraitレーダー、可能性マップ、候補バー、迷い・接戦カード、未探索候補カードを維持
- 「途中結果を見る」機能を維持し、途中結果では精度に関する注記を表示

質問数は時間予算で制御され、36問に達した時点で比較を終了します。

## v0.9 の改善点

- 初期画面を「標準版（10〜12分）」と「がっつり版（20〜30分）」の2モード化
- 標準版の回答・候補統計・trait・結果を保持したまま、追加分析としてがっつり版へ引き継ぐ機能を追加
- アンカー質問、カード仕分け、MaxDiff、接戦ペアワイズを組み合わせたハイブリッド探索フローを実装
- がっつり版に、安定と自律など8軸のトレードオフ比較を追加
- がっつり版の最後に、驚き・違和感・不安・次の行動を残す任意の内省質問を追加
- 履歴画面に「標準版」「がっつり版」「標準版 → がっつり版」のモードと関連を表示
- 保存形式をv0.9へ更新し、引き継ぎ元セッションIDと質問形式別ログを保存

標準版からの引き継ぎ分析では、標準版と同じ質問を繰り返さず、接戦候補・未探索候補・トレードオフを追加で確認します。

## v0.9.1 の改善点

- アンカー、カード仕分け、MaxDiff、ペアワイズ、トレードオフ、内省、避けたい要素の全形式に「戻る」を追加
- 回答ログを正として候補スコアとtraitを毎回再計算し、回答変更時の二重加算を防止
- MaxDiffのBEST・中間・WORSTから、重み付き疑似ペアワイズ比較を最大5本生成
- 候補スコアに比較回数ベースの信頼度補正と事前分布を追加
- 結果表示を0〜100の `displayScore` に統一し、少数回答の候補を50点付近へ補正
- raw traitを回答数で正規化し、レーダーチャート用スコアを5〜95に制限
- 結果に比較回数、信頼度、Best回数、Worst回数を表示

戻る操作では直前ログを取り消してから全スコアを再構築するため、回答を変更しても古い加点は残りません。
