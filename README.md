# Career Pairwise Lab（キャリア比較ラボ）

2択比較を通じて、キャリアの関心・判断軸・避けたい要素を整理するスマホ向けWebアプリです。適職を断定する診断ではなく、次に調べることや相談時の論点を作るためのツールです。

## 使い方

`index.html` をブラウザで開くだけで利用できます。外部API・外部ライブラリ・ビルド工程は不要で、GitHub Pagesにもそのまま配置できます。

## 主な機能

- 現在地とキャリア形態を選び、会社員・組織所属／個人活動のフローへ分岐
- 興味軸、業界・職種または活動ジャンル・収益モデル、仕事スタイルを2択比較
- 任意の比較理由と、避けたい要素・リスクを記録
- 結果を仮説としてランキング表示し、AI分析用プロンプトを生成・コピー
- 結果をブラウザの `localStorage` に最大30件保存し、履歴から再表示

入力内容は外部へ送信されません。保存データは利用中のブラウザ内にのみ保持されます。
